FOLDERS: 
- observations: folder with all the observational data to be 
analyzed (fits or txt, or both) from JADES and CANDELS catalogues.

FILES: 
- barred_gal_detection.ipynb: code that selects z sample and detects barred 
galaxies.
- cross_match.ipynb: code that computes the cross-match between JADES and 
CANDELS catalogues (DISCARDED). 
- Logbook.ipynb: logbook of the project
